## Lists
Lists are mutable objects

Common list functions: 
* append()
* insert()
* pop()
* remove()
* clear()
* reverse()
* sort() 

```python
A = [1,2,3]
B = A
B.append(4)
# This does not create a new list, it just assings B to the same list inside the memory
# To fix this you can cast B  'B = list(A)' or do 'B = A.copy()' 
>>> print(A)
[1,2,3,4] 
```
